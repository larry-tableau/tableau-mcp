How RLS Works

When a user authenticates via OAuth, Tableau Cloud returns their email as OAUTH_USERNAME. The MCP server substitutes this into UAT_USERNAME_CLAIM={OAUTH_USERNAME} and signs a UAT JWT with email: <user@example.com>.

Tableau Cloud validates the JWT against the registered public key, creates a scoped REST API session for that exact user, and enforces all Row-Level Security rules as if the user had signed in directly.

Each tool call requests only the minimum scopes needed (see table above). Tableau Cloud rejects requests that exceed the declared scopes in the JWT.
